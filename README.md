# 🌿 Healing Journey

> A mental wellness Android app that helps users track their mood, express themselves through art, and chat with an AI companion.

---

## About

Healing Journey is an Android app designed for emotional wellbeing. Users can log their daily mood with emoji check-ins, create expressive drawings, chat with an AI for support, and track their progress over time. Built with Java and Android Studio as a personal wellness companion.

---

## Features

- 😊 **Mood check-in** — pick an emoji to log how you feel each day
- 🎨 **Art drawing** — express emotions through a built-in drawing canvas
- 🤖 **AI chat** — talk to an AI companion for support and guidance
- 📊 **Progress tracking** — view your mood history and streaks
- 👤 **Profile** — edit your name, change your photo, view achievements
